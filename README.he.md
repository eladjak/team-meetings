# מנהל פגישות צוות

מערכת ניהול פגישות מודרנית עם React, FullCalendar, MUI, גרירה ושחרור לתזמון, ותמיכה מלאה בעברית RTL.

## תכונות

- לוח שנה אינטראקטיבי עם FullCalendar
- גרירה ושחרור לתזמון פגישות
- ממשק Material UI מודרני
- תמיכה מלאה ב-RTL עברית
- ניהול משתתפים

## טכנולוגיות

| טכנולוגיה | שימוש |
|-----------|-------|
| React | ספריית UI |
| FullCalendar | רכיב לוח שנה |
| Material UI | עיצוב ממשק |
| TypeScript | בטיחות טיפוסים |

## התקנה

```bash
git clone https://github.com/eladjak/team-meetings.git
cd team-meetings
npm install
npm start
```
