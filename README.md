# HaDerech Team Meetings 📅

מערכת לניהול פגישות צוות מתקדמת, נגישה ויעילה.

## תכונות עיקריות 🌟

- לוח שנה אינטראקטיבי
- תצוגות מרובות (יומי, שבועי, חודשי)
- ניהול משתתפים וחדרים
- התראות בזמן אמת
- נגישות מלאה
- תמיכה בעברית ו-RTL
- אינטגרציה עם יומנים חיצוניים

## התקנה 🚀

1. התקנת תלויות
npm install
הרצת סביבת פיתוח
npm run dev
בנייה לייצור
npm run build

## 🔧 קונפיגורציה
המערכת משתמשת בקבצי הגדרות הבאים:
- `constants.js` - הגדרות כלליות (חדרים, צוותים, וכו')
- `theme.js` - הגדרות עיצוב
- `.env` - משתני סביבה

## טכנולוגיות 💻

- React + Vite
- Material-UI
- Node.js + Express
- MySQL
- Socket.IO
- JWT
